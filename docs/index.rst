Getting Started
===============

`cborX` is a package hosted on PyPI.  I recommend installing with `pip
<https://pip.pypa.io/en/stable/>`_ into a `virtualenv
<https://hynek.me/articles/virtualenv-lives/>`_:

  pip3 install cborx

The `overview` will give you some simple examples of usage.

The `API Reference` will give you a comprehensive explanation of ``cborX``'s API and
features.
